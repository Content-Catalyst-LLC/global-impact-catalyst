# Global Impact Catalyst Demo Plugin

Shortcode:

```text
[global_impact_catalyst_demo]
```

This plugin renders a browser-based demo for creating a traceable impact measurement record. The demo is client-side and does not submit visitor inputs to Sustainable Catalyst.

Suggested page wrapper:

```html
<div class="scgic-demo-shell">
  [global_impact_catalyst_demo]
</div>
```
