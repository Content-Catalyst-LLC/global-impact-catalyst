# Localization and Language Governance

v1.10.0 seeds English (United States), Spanish, and French interface strings and supports workspace-specific locale definitions.

Locale records include:

- BCP 47-style locale code;
- display name;
- left-to-right or right-to-left direction;
- fallback locale;
- active status; and
- translation dictionary.

Localization changes labels and explanatory language only. Canonical identifiers, numeric values, units, hashes, evidence links, validation results, and calculation semantics remain language-independent.
