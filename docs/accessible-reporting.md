# Accessible Reporting

Global Impact Catalyst v1.8.0 treats accessibility metadata as part of the reporting contract.

Generated HTML includes a declared language, UTF-8 encoding, viewport metadata, a skip link, one primary heading, ordered section headings, semantic definition lists, and structured references. Dashboard cards require text alternatives. Templates record an accessibility profile with a default target of WCAG 2.2 AA, valid heading order, table captions, and text alternatives.

The generated HTML is intentionally self-contained and does not require JavaScript to read. Institutions remain responsible for testing final branded templates, linked documents, charts, color contrast, and assistive-technology behavior in their publishing environment.
