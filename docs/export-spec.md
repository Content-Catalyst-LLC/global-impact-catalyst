# Export Specification — v1.0.1

Global Impact Catalyst exports UTF-8 JSON records governed by `schemas/global_impact_record.schema.json` and optional Markdown briefs generated from the same in-memory record.

## JSON

- Record type: `global_impact_catalyst_record`
- Timestamp: RFC 3339 / JSON Schema `date-time`
- Optional beneficiaries and budget: `null` when omitted; explicit zero remains zero
- Derived metrics: fully defined under `metrics`
- Quality signal: total plus four transparent components
- Traceability path and boundaries: fixed canonical arrays in v1.0.1

## Reproducibility

The CLI accepts `--generated-at` so fixtures, examples, and releases can be regenerated byte-for-byte except where formatting tools intentionally differ.

## Markdown

The Markdown brief includes record metadata, derived metrics, quality components, interpretation notes, method notes, and boundary statements. It labels the quality score as a completeness/review signal.
