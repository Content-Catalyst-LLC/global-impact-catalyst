# Export Specification

Global Impact Catalyst exports records as JSON, with an optional Markdown brief.

## JSON record

The JSON record contains:

- Record metadata
- Initiative and goal
- SDG-style theme
- Indicator, unit, baseline, current value, and target
- Source and method notes
- Confidence and review status
- Computed metrics
- Interpretation notes
- Boundaries and traceability path

See `schemas/global_impact_record.schema.json` for the current schema.

## Markdown brief

The Markdown brief is intended for review and documentation. It is not a formal assurance report.
