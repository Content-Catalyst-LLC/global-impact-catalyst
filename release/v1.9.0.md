# Global Impact Catalyst v1.9.0

## Public API, Embeds, and Sustainable Catalyst Handoffs

This release adds the controlled exchange layer for Global Impact Catalyst. Approved publication snapshots can be exposed through privacy-safe public APIs and read-only embeds. Authenticated clients can read scoped workspace resources and create checksum-bound handoff packages for Sustainable Catalyst products.

### Added

- SQLite migration 10 and integration repository v1.9.0.
- API clients, scopes, one-time key issuance, hashed key storage, revocation, fixed-window rate controls, idempotency, and access logs.
- Public catalog and publication profile endpoints restricted to approved current snapshots.
- Paginated workspace API resources and JSON-LD metadata.
- Five governed embed types and public rendering.
- Ten Sustainable Catalyst destination handoffs, delivery receipts, and event history.
- Integration schemas, examples, CLI commands, service methods, WordPress REST routes, Integration Hub, and public shortcodes.
- Workspace export and idempotent lossless restore of integration metadata without plaintext key material.

### Compatibility

The canonical impact contract remains v1.1.0. Evidence, registry, measurement, review, analysis, and reporting formats remain at v1.3.0 through v1.8.0. Existing v1.8.0 databases migrate in place to database schema 10.
