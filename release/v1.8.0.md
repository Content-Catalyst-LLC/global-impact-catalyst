# Global Impact Catalyst v1.8.0

## Reporting, Publication, and Reproducible Export Studio

This release adds governed report templates and documents, citations, methodology appendices, accessible HTML and Markdown, dashboard definitions, publication snapshots, deterministic export bundles, checksum verification, workspace export and restore, SQLite migration 9, CLI operations, and an authenticated WordPress Reporting Studio.
