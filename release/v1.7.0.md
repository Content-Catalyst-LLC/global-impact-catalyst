# Global Impact Catalyst v1.7.0

## Trends, Comparisons, Scenarios, and Uncertainty

This release adds database migration 8, a versioned analysis repository, governed trend statistics, benchmark and comparison-set controls, constant/linear/compound/step scenarios, target trajectory gaps, uncertainty intervals, sensitivity analysis, persisted analytical runs, CLI commands, schemas, examples, and an authenticated WordPress Analysis Studio. The canonical impact contract remains v1.1.0; review workflow remains v1.6.0. Analytical output is descriptive and conditional, not assurance, certification, causal proof, or a verified forecast.
