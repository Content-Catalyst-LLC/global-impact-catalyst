# Global Impact Catalyst v1.2.0

## Persistent Initiatives, Portfolios, and Measurement Repository

v1.2.0 turns generated contracts into reopenable, revisable workspaces while preserving the v1.1.0 canonical contract and cross-runtime calculation guarantees.

### Added

- SQLite repository abstraction with three tested schema migrations.
- Complete canonical contract snapshots plus indexed workspace, initiative, goal, indicator, observation, target, and source projections.
- Shared application service for create, save, autosave, import, duplicate, list, export, and restore use cases.
- Initiative search, lifecycle filtering, archive, restore, and duplication.
- Portfolio records and ordered initiative memberships.
- Explicit saved revisions and recoverable autosave drafts.
- Optimistic concurrency errors carrying expected and current revisions.
- Idempotent v1.0.x and v1.1.0 imports with audit receipts.
- Portable workspace export and repeatable restore bundles.
- Transactionally consistent SQLite backups.
- Persistent authenticated WordPress workspace shortcode and REST repository.
- Repository CLI for local and institutional workflows.

### Compatibility

- The canonical contract and schema version remain v1.1.0 because persistence does not change the data contract.
- The Python and browser calculation engines remain exact-parity v1.1.0 engines.
- The package, application service, WordPress plugin, workspace bundle, and release version are v1.2.0.
- `[global_impact_catalyst_demo]` remains available unchanged; `[global_impact_catalyst_workspace]` is additive.
